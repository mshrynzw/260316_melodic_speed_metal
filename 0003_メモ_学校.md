# 学年の流れ

一般的なスケジュールはこうです。

```txt
8月中旬　秋学期開始
12月中旬　クリスマス休暇

1月上旬　春学期開始
5月末　　学年終了
```

つまり

学校は8月に始まり、5月に終わる

です。

## 秋学期

期間

```txt
8月〜12月
```

特徴

- 新学年スタート
- 新しいコース履修開始
- 12月にクリスマス休暇

フィンランドでは、クリスマスがかなり大きな区切りになります。

## 春学期

期間

```txt
1月〜5月
```

特徴

- コース履修続き
- 期末試験
- 卒業試験

高校3年生は、Matriculation Exam（大学入試試験）があります。

# コース

- 1回の授業 = 約75分
- 1コース = 約6〜7週間
- 1コースの授業回数 = 約18〜20回
- コース登録は年に2回
  - 8月に秋学期のコース選択
  - 12月〜1月に春学期のコース選択
- ただし変更は可能
  - コース変更
  - 別の授業を追加
  - 授業を外す
- 科目ごとに「順番（前提条件）」がある

## 例：英語

```txt
英語1　基礎
英語2
英語3
英語4
英語5
英語6　大学入試レベル

英語7　上級
英語8　専門
```

英語5までは必須。

## 具体例

> 秋学期に英語１，２，３，をコース登録したけど、英語１の最終試験（？）で落ちてしまい、英語２を受けられないケースってある？

はい、**普通にあり得ます。**
ただしフィンランドの高校では、日本のように「完全に次の授業が受けられない」というより、**いくつか救済の仕組みがあります。**

整理するとこうです。

---

### 基本ルール

原則

```text
英語1 → 合格
↓
英語2
```

つまり

**英語1に合格していないと英語2は正式には履修できません。**

---

### ただし実際の運用

フィンランドの高校はかなり柔軟で、次のようなケースがあります。

#### ① 再試験（かなり一般的）

コース終了後に

```text
再試験（retake exam）
```

があります。

つまり

```text
英語1
↓
試験落ちる
↓
再試験
↓
合格
↓
英語2
```

---

#### ② 次のコースを仮履修

学校によっては

```text
英語1　落ちた
↓
英語2　一応受ける
↓
英語1を後で再試験
```

ということもあります。

---

#### ③ 次のコースを取れない

もし再試験でもダメなら

```text
英語1
↓
もう一度履修
```

になります。

## 秋学期と春学期の違い

実際の学校ではだいたいこうなります。

| 秋学期               | 春学期          |
| ----------------- | ------------ |
| 基礎科目が多い           | 発展科目が増える     |
| 新学年の導入            | 応用           |
| クラスメイトがまだ固定されていない | 友達グループができている |

# ロヴァニエミの高校の規模イメージ

実際の例（Ounasvaara高校）

* 学生：約800人
* 新入生：約250人/年
* 教員：約30人
* カウンセラー：3人
* 特別支援教師：2人
* 心理士・ソーシャルワーカーあり ([Oukku][1])

フィンランドでは **生徒サポートのスタッフが多い**のが特徴です。
