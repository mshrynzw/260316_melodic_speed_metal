# アニメタイトル
「めろでぃっく・すぴーど・めたる！」

# 意味
メロディックスピードメタルというジャンルが確立されていない状況で、自分たちがメロディックスピードメタルというジャンルを確立する意味

# 作品コンセプト
1990年代、フィンランド北部の街ロヴァニエミ。  
天才ドラマーのカーリナは、気に入らないとバンドメンバーをすぐクビにしてしまう。  
平穏な日常に物足りなさを感じていた凛は、半ば強制的にキーボードとしてバンドに加入させられる。  
そこへ歌のうまい転校生アイナが現れ、4人のバンドが動き始める。  
彼女たちはまだ存在しない音楽――メロディックスピードメタルを作ろうとしていた。

# 舞台
フィンランド・ロヴァニエミ

# あらすじ
フィンランドのオーロラの街にある高校。  
カーリナはバンドをやっていたが、気に入らないとクビにして、また新しいメンバを探す。  
そんなクラスメートを横目に平穏な高校生活を過ごしていた凛。何か日常に物足りなさを感じ始めた凜。  
そんなある日、歌がうまい（＝カリーナが納得できる最低限のレベル）アイナが転校してきたことを切っ掛けにバンドメンバーがそろう。  
カーリナが納得できる曲を作りあげることを目指していく。

# バンド

Melospe（メロスピ。当時メロディックスピードメタルというジャンルが確立されていない状況で、自分たちがメロディックスピードメタルというジャンルを確立する意味合いがある）
## ジャンル
メロディックスピードメタル

# メインキャラクター（4人）
## キーボード（主人公）
白石 凛（しらいし りん）  
フィンランドのロヴァニエミ出身。小さいころからクラシックのピアノが弾ける。父親はアマチュアのオーケストラの指揮者。  
メロディックスピードメタルにはキーボードが必要でピアノが弾ける凜に目を付けたカーリナに半ば強制的にバンドに加入する。  
加入してからわかったことだが、バンドで全くみんなと合わない演奏をしてしまい、へこむ。  
平穏な日々に何か物足りなさを感じ始めていて、それをバンド活動の中で見つけていこうとする。  
口調：普通

## バンドリーダー・ドラム
カーリナ・ハンセン
フィンランド・ロヴァニエミ出身。メンバーに無茶ぶりを繰り返す。破天荒。  
幼いころに親に捨てられ施設で育った過去を持つ。  
リリアとは幼馴染もとい、子分としてこき使う。  
幼いころ、日本のとあるメタルバンドに衝撃を受け、ドラムをたたくようになる。  
凛に定期的に日本のCDを輸入してもらうようにしている。  
エピソード「カレーが辛いからリハーサルから帰った」  
コミュニケーションが苦手で人づきあいがリリアしかいない。  
ライブはそこまで重視していないようで、観客は「曲を客観的に評価するための実験装置」だと思っている。  
天才肌。作曲・作詞・編曲などができ、他の楽器もある程度はできる。幼いころ、リリアにベースを拷問のように教えていた。  
ただし、自分自身の声色には自信がなく、その分ボーカル選びにこだわりが強い。  
たまたま凜の家にあったオーケストレーションの教科書を一読しただけで、オーケストラの編曲もできるようになった。  
表には出さないが、作品への執念と孤独のつらさに挟まれている。  
口調：冷たい丁寧語

## ベース
リリア・ルンド
フィンランド・ロヴァニエミ出身。ベースをかなり弾ける。人前で演奏するのは苦手だが 音楽は大好き。  
カリーナと同じく施設で育ったため幼馴染だが、カリーナの言うことには逆らえない。  
幼いころ、カーリナに聞かせてもらった日本のとあるメタルバンドに同じく衝撃を受け、強制的にカーリナからベースを教わった。  
自分自身は何も才能ないと思い込んでいるが、カーリナは自分よりも呑み込みが良いことに気が付き、カーリナが納得できる演奏技術を持つようになっていた。  
ただし、ベースがうまいのはカーリナの猛特訓してくれたおかげで、「演奏技術＝友情の結晶」と思い込んでいる。  
周りからは幼馴染の友情が一周して、ヤンデレのよう映っている。  
バンドも辞めたいが辞めさせてもらえない。  
表には出せないが承認欲求があり作詞をするが、毎回カーリナに没にされる。
北欧の自然が好きで、幻想的なオーロラが好きで、のちにシンセサイザーの幻想的な音をバンドを取り入れるきっかけになる。  
性格
極度内気・コミュ障・やさしい
口調：気弱な丁寧語

## ギター・ボーカル
アイナ・エーク
転校生。フィンランド・ロヴァニエミ出身ヘルシンキ育ち。ヘルシンキの中学まで寄宿舎で生活していたが、幼いころの思い出のオーロラのもとで生活したくて、実家のロヴァニエミに戻った。  
実家は広大な敷地を持つお屋敷。  
ロヴァニエミのオーロラに感動し、同じく北欧の自然が好きなリリアと共感することが多い。  
歌はうまい（＝カリーナが納得できる最低限のレベル）が、ギターは初心者。  
また、英語の発音が苦手で、カーリナのディレクションに何とか応えようとしている。  
人間観察が得意でコミュニケーション能力も高い。バンドリーダーはカーリナだが、彼女は毎回無理難題をメンバーに押し付けるため、後々愛奈が実質的なバンドのまとめやく役になる。  
実は初めて会った時から、カーリナが孤独に苛まれていることに気が付いている。  
性格  
まとめ役・やさしい  
明るい  
口調：明るい

# 作風（きらら系）
カーリナの殺伐さはあるが、基本はゆるい日常・バンド練習。  
フィンランド・ロヴァニエミの町の生活。  
時々、試行錯誤しながら小さなライブ。  
目標はカーリナが納得できる曲を作り上げること。

# アイディア
- 幼いころ。リリア「もう3日も寝ずにベースを弾き続けてるよ。お願いします。もう休ませてください！」カリナ「練習を止めるなら、友達の縁を切ります」リリア「頑張らさせてください！」
- 転校前。夜の街で。アイナ「幻想的なオーロラですね」凜「そうなのかな。もう見慣れてるから」  
- カーリナは実家が裕福なアイナに目をつけ、様々な機材を買ってもらっていたが、アイナは「このまま買い与えてしまい続けると、カーリナの孤独をさらに深めてしまう」と考え改め、持ち前のコミュニケーション能力でカーリナの頼みをうまく断り続けることにした。
- カーリナ「ご両親にお願いしてレコディングスタジオやミキシングスタ時を建ててください」アイナ「えっ？建物を新しく建てるっていうこと？聞いてみるけど、たぶん無理かな。無理だね」
- リリア「もうバンド辞めたいよ！」カーリナ「辞めません。辞めるなら、友達の縁を切ります」リリア「頑張らさせてください！」  
- リリア「カーリナが教えてくれたベースだから……」カーリナ「普通に練習しただけです」凛「いや拷問だったって言ってたよ」
