# カイサ・ニエミネン（Kaisa Nieminen）
ドラム担当
ビジュアル：無表情、青い目、前髪はアシンメトリーで分け、ロングストーレートの毛先が軽くウェーブしている。もともと金髪だが、とある日本のバンドにあこがれて黒染めしている。

# エリナ・ランタネン（Elina Rantanen）
ベース担当
ビジュアル：ネガティブな表情。青い目。髪色はダークブラウン。前髪は長めにして片目が隠れていて、ロングの三つ編み。ただし不器用なので、三つ編みはいつも整っていない。

# ハンナ・カッリオ（Hanna Kallio）
