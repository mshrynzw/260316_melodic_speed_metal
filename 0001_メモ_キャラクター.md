# 白鳥 桔梗（しらとり ききょう）
キーボード担当（初期はピアノのみ、のちにシンセサイザーも担当するようになる）  
一人称：私  
口調：普通  
ヴィジュアル：純日本人、髪型はロングの日本和風のお姫様カット
```
フィンランドのロヴァニエミ出身。小さいころからクラシックのピアノが弾ける。父親はアマチュアのオーケストラの指揮者。  
メロディックスピードメタルにはキーボードが必要でピアノが弾ける凜に目を付けたカイサに半ば強制的にバンドに加入する。  
加入してからわかったことだが、バンドで全くみんなと合わない演奏をしてしまい、へこむ。  
平穏な日々に何か物足りなさを感じ始めていて、それをバンド活動の中で見つけていこうとする。 
```

# カイサ・ニエミネン（Kaisa Nieminen）
ドラム担当  
性格：天才肌・破天荒  
口調：冷たい丁寧語  
ビジュアル：無表情、青い目、前髪はアシンメトリーで分け、ロングストーレートの毛先が軽くウェーブしている。もともと金髪だが、とある日本のバンドにあこがれて黒染めしている。  
```
フィンランド・ロヴァニエミ出身。メンバーに無茶ぶりを繰り返す。破天荒。  
幼いころに親に捨てられ施設で育った過去を持つ。  
エリナとは幼馴染もとい、子分としてこき使う。  
幼いころ、日本のとあるメタルバンドに衝撃を受け、ドラムをたたくようになる。  
桔梗に定期的に日本のCDを輸入してもらうようにしている。  
エピソード「カレーが辛いからリハーサルから帰った」  
コミュニケーションが苦手で人づきあいがエリナしかいない。  
ライブはそこまで重視していないようで、観客は「曲を客観的に評価するための実験装置」だと思っている。  
天才肌。作曲・作詞・編曲などができ、他の楽器もある程度はできる。幼いころ、エリナにベースを拷問のように教えていた。  
ただし、自分自身の声色には自信がなく、その分ボーカル選びにこだわりが強い。  
たまたま凜の家にあったオーケストレーションの教科書を一読しただけで、オーケストラの編曲もできるようになった。  
表には出さないが、作品への執念と孤独のつらさに挟まれている。  
```

# エリナ・ランタネン（Elina Rantanen）
ベース担当  
性格：極度内気・コミュ障・密かに承認欲求が強い・不器用だがやさしい  
口調：怯えた丁寧語  
ビジュアル：ネガティブな表情。青い目。髪色はダークブラウン。前髪は長めにして片目が隠れていて、ロングの三つ編み。ただし不器用なので、三つ編みはいつも整っていない。
```
フィンランド・ロヴァニエミ出身。ベースをかなり弾ける。人前で演奏するのは苦手だが 音楽は大好き。  
カリーナと同じく施設で育ったため幼馴染だが、カリーナの言うことには逆らえない。  
幼いころ、カイサに聞かせてもらった日本のとあるメタルバンドに同じく衝撃を受け、強制的にカイサからベースを教わった。  
自分自身は何も才能ないと思い込んでいるが、カイサは自分よりも呑み込みが良いことに気が付き、カイサが納得できる演奏技術を持つようになっていた。  
ただし、ベースがうまいのはカイサの猛特訓してくれたおかげで、「演奏技術＝友情の結晶」と思い込んでいる。  
周りからは幼馴染の友情が一周して、ヤンデレのよう映っている。  
バンドも辞めたいが辞めさせてもらえない。  
表には出せないが承認欲求があり作詞をするが、毎回カイサに没にされる。  
ミュージシャンとしてのヴィジュアル面にこだわりがあり、お金持ちで豪邸に住んでそこでインタービューを受ける将来を勝手に描いている。  
北欧の自然が好きで、幻想的なオーロラが好きで、のちにシンセサイザーの幻想的な音をバンドを取り入れるきっかけになる。 
```

# ハンナ・カッリオ（Hanna Kallio）
ギター・ボーカル担当  
一人称：わたし  
性格：まとめ役・やさしい・明るい  
口調：明るい  
ビジュアル：穏やかな表情。緑の目。髪色は金髪。ミディアムで動きのあるナチュラルウェーブ。
```
転校生。フィンランド・ロヴァニエミ出身ヘルシンキ育ち。ヘルシンキの中学まで音楽の芸術学校の寄宿舎で生活していたが、幼いころの思い出のオーロラのもとで生活したくて、実家のロヴァニエミに戻った。  
実家は広大な敷地を持つお屋敷。  
ロヴァニエミのオーロラに感動し、同じく北欧の自然が好きなエリナと共感することが多い。  
歌はうまい（＝カリーナが納得できる最低限のレベル）が、ギターは初心者。  
また、英語の発音が苦手で、カイサのディレクションに何とか応えようとしている。  
人間観察が得意でコミュニケーション能力も高い。バンドリーダーはカイサだが、彼女は毎回無理難題をメンバーに押し付けるため、後々愛奈が実質的なバンドのまとめやく役になる。  
実は初めて会った時から、カイサが孤独に苛まれていることに気が付いている。
```
